# UI重新设计计划 - 高级扁平化三栏布局

## 设计目标
创建一个现代化、扁平化、高级感的三栏布局界面

## 布局结构

```
┌─────────────────────────────────────────────────────────────────────────┐
│ 📸 照片旅行地图                                             [设置] │
├─────────────────────────────────────────────────────────────────────────┤
│ 左侧功能栏      │ 中间地图区域          │ 右侧照片墙      │
│ (280px)        │ (自适应)            │ (320px)        │
│                │                     │                │
│ 📤 上传照片    │                     │ 🖼️ 照片墙      │
│                │      🗺️ 地图        │                │
│ 🏷️ 标签筛选    │                     │                │
│                │                     │                │
│ ⚙️ 其他设置    │                     │                │
│                │                     │                │
│                │                     │                │
└─────────────────────────────────────────────────────────────────────────┘
```

## 详细设计

### 1. 左侧功能栏 (280px)
- **上传照片**
  - 简洁的拖拽上传区域
  - 进度条显示
  - 批量上传支持
  - 手动补充信息对话框

- **标签筛选**
  - 标签列表展示
  - 点击筛选功能
  - 标签数量统计
  - 添加/编辑/删除标签

- **其他设置**
  - 轨迹显示开关
  - 数据备份/恢复
  - 清空数据
  - 关于信息

### 2. 中间地图区域 (自适应)
- **地图容器**
  - 全屏地图显示
  - 高德地图瓦片
  - 照片标记点
  - 轨迹线显示
  - 地图控制按钮（缩放、定位）

### 3. 右侧照片墙 (320px)
- **照片列表**
  - 网格布局展示
  - 缩略图预览
  - 照片信息（时间、地点、标签）
  - 点击查看大图
  - 滚动浏览

## 设计风格

### 颜色方案
- **主色调**：蓝色系 (#3b82f6)
- **背景色**：浅灰 (#f3f4f6)
- **卡片背景**：纯白 (#ffffff)
- **文字色**：深灰 (#1f2937)
- **边框色**：浅灰 (#e5e7eb)

### 扁平化设计
- 去除阴影效果
- 使用细边框分隔
- 圆角统一为 8px
- 按钮使用纯色背景
- 图标使用线性风格

### 交互效果
- 悬停时背景色变化
- 点击时轻微缩放
- 平滑过渡动画
- 响应式布局

## 实现步骤

### 阶段3：中间地图区域

## 技术要点

### 布局技术
- Flexbox 三栏布局
- 固定左右栏宽度
- 中间栏自适应
- overflow 控制滚动

### 组件拆分
- 左侧：Sidebar 组件
- 中间：MapSection 组件
- 右侧：PhotoWall 组件

### 状态管理
- 使用现有的 AppContext
- 添加照片墙显示状态
- 优化筛选逻辑

### 性能优化
- 照片懒加载
- 虚拟滚动
- 防抖处理
